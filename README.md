# API-Hotelaria
API REST com JSON Server 

## Clone o repositório do GitHub no Git
$ git clone <https://github.com/IgorFerrer053/API-Hotelaria>

## Instale as dependências
$ npm install

## Execute o script
$ npm run start

## Acesse o servidor local
$ <http://localhost:3000>
